== Installation ==


= From WordPress Dashboard =

1. Plugins > 'Add New' in the plugins dashboard
2. Search  'Content Permission for Pages & Posts.
3. Click 'Install' Button
4. Activate the plugin on the Plugin dashboard. No Setting Pages.

= Upload Zip file in Dashboard =

1. Download Zip file of plugin and Hit 'Add New' in the plugins dashboard
2. Click 'Upload' Button on left top.
3. Select `rcd-right-click-disabler.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `rcd-right-click-disabler.zip`
2. Extract Zip file `rcd-right-click-disabler.zip` directory to your computer
3. Upload the `rcd-right-click-disabler` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
 
== Frequently Asked Questions ==
= How can i Disable Right Click/Stop Content Copy ? =

All you have to activate the plugin.Once after the Installation & Activation of plugin it does all this for you.
 
= Is Right Click Disabled for Admins Too ? =
No, As per the users demand we have removed restriction for admin role to ease them to copy their content while development.

= Can i Disable it on Few pages ? =
No, RCD is activated though out the website and currently this functionality isn.t available.
